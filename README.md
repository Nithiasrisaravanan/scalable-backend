
# Healthcare Backend API (Spring Boot)

## Tech Stack
- Java
- Spring Boot
- REST APIs
- Maven

## Features
- Patient management system
- CRUD APIs
- Clean architecture (Controller-Service-Model)

## Run

### Prerequisites
- Java 17+
- Maven

### Steps
mvn clean install
mvn spring-boot:run

## Endpoints
GET /patients
POST /patients
DELETE /patients/{id}
