
- # 🏥 Healthcare Backend API System (Spring Boot)

A scalable backend system built using **Java Spring Boot** to simulate healthcare data management workflows. This project demonstrates clean architecture, REST API design, and backend system reliability — aligned with real-world healthcare technology systems.

---

## 🚀 Tech Stack

- Java (JDK 17+)
- Spring Boot
- Maven
- REST APIs
- OOP & Clean Architecture

---

## ✨ Features

- Patient data management system  
- RESTful API endpoints (CRUD operations)  
- Modular architecture (Controller → Service → Model)  
- Scalable and maintainable backend design  
- Error handling and structured request processing
